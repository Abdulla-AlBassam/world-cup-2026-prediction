# World Cup 2026 Prediction

Predict every match of the 2026 World Cup, then share the link.

**Live: [world-cup-2026-prediction-kappa.vercel.app](https://world-cup-2026-prediction-kappa.vercel.app)**

No signup. No tracking. The whole bracket is encoded into the URL, so sending the link is the share.

## What you can predict

- Final ranking of all 12 groups
- Which 8 third-placed teams advance
- Every knockout match through the final
- Golden Ball, Golden Boot, Golden Glove, Best Young Player, Fair Play Trophy

## Local development

```bash
npm install
npm run dev
```

Open [localhost:3000](http://localhost:3000).

## Stack

Next.js, TypeScript, Tailwind, shadcn/ui, framer-motion, lz-string, html-to-image, flag-icons.
